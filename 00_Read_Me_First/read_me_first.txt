How to use the software

1. Unzip the folder and place it in the htdocs folder located inside xampp folder (localhost)
2. start apache and mysql from the xampp controlo panel
3. Type in localhost/phpmyadmin in the address bar of your browser
4. Create a database named help_desk
5. Import database table from the database folder located inside the project folder
6. Make sure your system is connected to the internet for the software to function well
   Because there are some libraries that need to be fetched online.
7. watch the video tutorial
